# Aurion Desktop Environment Alpha

This directory contains the first complete AurionOS desktop-environment layer.

It is intentionally installed as an alpha layer on top of the stable Lubuntu/LXQt
base. The default ISO session remains conservative, while these commands provide
native Aurion surfaces for the ten DE blocks:

- `aurion-window-manager`
- `aurion-de-shell --mode topbar`
- `aurion-de-shell --mode dock`
- `aurion-launcher`
- `aurion-settings`
- `aurion-files`
- `aurion-notifications`
- `aurion-login-preview`
- `aurion-startdesktop`
- packaging metadata under `packaging/aurion-desktop-environment/`
