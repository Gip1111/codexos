# Aurion Store Foundation

Aurion Store is the future graphical app installation layer.

Alpha policy:

- no terminal required in the final product
- Flatpak-first
- Snap optional for MVP
- `.deb` support for trusted local packages
- `.flatpakref` support through graphical flow
- AppImage integration flow: run once, integrate, add launcher

The alpha command lists a catalog and explains the install strategy. It does not install packages yet.
