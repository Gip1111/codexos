# Aurion Experience

Aurion Experience is the alpha graphical control surface for AurionOS.

It is a local HTML/CSS/JS application for the first ISO because that avoids adding fragile live image dependencies. The target implementation is Qt6/QML with the same task model:

- ask for a task in natural language
- get a clear next action
- open browser/email paths
- start app installation flows
- inspect hardware and diagnostics
- keep AI local-first by default
