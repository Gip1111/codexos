AurionOS Alpha

Fast Track v0.1 on the Lubuntu 24.04 LTS base. This live system validates boot, installability, branding, diagnostics, AI foundation, and hardware compatibility direction.

System

Run aurion-status for the live validation summary.

Hardware

Run aurion-hardware-center for read-only compatibility classification.

AI

Run aurion-ai-status. The future default is local Ollama with phi4-mini.

Apps

Run aurion-store to inspect the alpha catalog and final install strategy.

Alpha safety

  • Ubuntu signed boot components are preserved.
  • No custom kernel is used.
  • No proprietary or DKMS modules are installed by the live ISO.
  • Cloud AI is disabled by default and must remain opt-in.
