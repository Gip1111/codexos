# AurionOS Diagnostics

Alpha diagnostics are read-only tools intended to prove the OS assistant and hardware center direction without risking boot stability. On the desktop, `aurion-diagnostics` opens a graphical surface; use `aurion-diagnostics --cli` for the terminal report.

Included commands:

- `aurion-diagnostics`
- `aurion-diagnostics-gui`
- `aurion-hw-scan`
- `aurion-rollback-status`
